Curso de Especialização à Distância em Desenvolvimento de Sistemas Web e Mobile (EspWebMob)

O curso de especialização em Desenvolvimento de Sistemas Web e Mobile é 100% À DISTÂNCIA.

Atualmente o DIN é responsável por dois cursos de graduação da UEM: Ciência da Computação (integral) e Informática (noturno), além da ênfase de software do curso de Engenharia de Produção.

O DIN também oferece três cursos de pós-graduação: Especialização em Desenvolvimento de Sistemas para Web e Mobile e Mestrado e Doutorado em Ciência da Computação.




Histórico

Este curso teve início em 2002 com o título "Tecnologia em Desenvolvimento para Web" de forma presencial. Nesta turma houve 31 matriculados oriundos de diversas cidades do Paraná e sul de São Paulo. Destes, 19 concluiram monografia e obtiveram o certificado de conclusão do curso.

Em 2003 realizou-se uma reformulação pedagógica e definiu-se enfoque em desenvolvimento de sistemas de informação para Web. Efetuou-se melhorias nos laboratórios e implementou-se mais aulas práticas. Em agosto/2003 iniciou-se as aulas da segunda turma com 33 alunos matriculados vindos das cidades de Campo Grande, Campo Mourao, Castro, Goioere, Mandaguari, Maringá, Martinópolis, Palmitópolis, Presidente Prudente e Umuarama. Destes, 30 concluiram os créditos em disciplinas e atualmente estão desenvolvendo suas monografias, com previsão para término em abril/2005, para conclusão do curso.

Desde então, o curso teve 15 turmas, todas no formato presencial.

Em 2024, o curso migrou para os moldes à distância (100% remoto), buscando ampliar a participação e a formação de mão de obra especializada para o Brasil e o exterior.



Disciplinas do Curso

O curso é composto de 380 horas/aulas à distância que abangem as seguintes disciplinas:

  • Introdução ao Ensino à Distância (30h) - Ambiente Virtual de Ensino-Aprendizagem. Ferramentas para navegação e busca na Internet. Metodologias de estudo baseadas nos princípios de autonomia, interação e cooperação.
  • Metodologia de Pesquisa em Computação (40h) - Tipo de conhecimento; conceito de realidade; classificação das ciências; pesquisa científica; divulgação da pesquisa científica; experimentos controlados em computação.
  • Programação Orientada a Objetos (30h) - Conceitos de programação orientada a objetos, correlacionando elementos teóricos com a prática.
  • Modelagem de Software (20h) - Modelagem de software orientado a objetos. Métodos de análise e de projeto de software. Notação de modelos orientados a objetos. Ferramentas para modelagem de software orientado a objetos.
  • Desenvolvimento Front-End com HTML e CSS (60h) - Conceitos, Sintaxe e Elementos da Linguagem HTML, seus respectivos atributos e aplicações. Conceitos, Sintaxe e propriedades da Linguagem CSS, suas unidades de medida e aplicações. Relações entre CSS e HTML, incluindo Design Responsivo, uso de Framework e Bibliotecas e componetização.
  • Interação Humano-Computador e User eXperience (20h) - Fundamentos teóricos e práticos da Interação Humano-Computador. Modelos e padrões para a prototipação de interfaces web. Projeto de interface com Experiência do Usuário.
  • Desenvolvimento para Dispositivos Móveis (80h) - Visão geral sobre mobilidade e tipos de dispositivos móveis. Tecnologias e ferramentas para o desenvolvimento de aplicativos móveis para as plataformas Android e iOS.
  • Desenvolvimento Front-End com Javascript e Frameworks (50h) - Linguagem Javascript: Declaração de variáveis e tipos; Estruturas e funções; Classes e Modularização; Objetos da Linguagem; Principais APIs e objetos do Navegador; Design Patterns. Introdução a TypeScript. Ferramentas: análise de qualidade de código, agrupador de módulos e transcompilador. Bibliotecas e frameworks para criação de interfaces de usuário.
  • Desenvolvimento Web Back-end (50h) - Conceitos de desenvolvimento de aplicações para Web Backend. Desenvolvimento de aplicações Web utilizando API REST. Integração de aplicações Web com banco de dados relacional e não relacional. Frameworks para o desenvolvimento de aplicações Web.
